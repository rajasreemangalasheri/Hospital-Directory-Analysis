# Hospital Directory Analysis using Power BI

## Project Overview

This project analyzes the Hospital Directory Dataset using Microsoft Power BI to provide insights into hospital distribution across India.

## Objectives

- Analyze hospital distribution
- Compare hospital categories
- Visualize hospital care types
- Create an interactive dashboard

## Tools Used

- Microsoft Excel
- Power BI Desktop
- Power Query
- DAX

## Dashboard Features

- KPI Cards
- Top 10 States by Hospitals
- Top 10 Districts by Hospitals
- Hospital Category Distribution
- Hospital Care Type Distribution
- Interactive Slicers
